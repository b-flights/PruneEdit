# PruneEdit
Edit notes and documents like a bonsai tree

## About
PruneEdit is a tool that allows you to instantly edit multiple versions of a text document using an interactive tree visual. Each version of your text, along with an optional description, is represented by a node on the tree. You can quickly select different versions and edit their contents.

## Running
Requirements:

`Python` (3.13 or later) https://www.python.org/downloads/<br>
`wxPython` https://www.wxpython.org/pages/downloads/<br>
`mypy` (Optional, for type checking)
