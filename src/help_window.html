PruneEdit

PruneEdit is a tool that allows you to instantly edit multiple versions of a text document using an interactive tree visual. Each version of your text, along with an optional description, is represented by a node on the tree. You can quickly select different versions and edit their contents.


Tree Panel Usage

Text versions are selected by clicking on the corresponding node in the tree panel. Hovering over a node displays its description in the top left of the panel. The contents and description of the current node (highlighted in red) can be edited in the edit boxes to the left of the tree panel.


Operations

Show Tree/Hide Tree

Display/hide the tree panel.

Update

Update the content and description of the current version.

Save as New

Save the content and description as a child version of the current node.

New Blank Version

Add a blank version as a child of the current version.

Set as Main

Set the current version as the root version, discarding all prior versions.

Delete

Delete the current version and all child versions.

Delete Changes

Delete all child versions of the current version only.


File Menu

New

Create a new empty version tree.

Open Document

Load in a .txt document file as a node in a new tree.

Save Document

Save the current node's content as a .txt file.

Open Tree

Load a version tree saved as a .json file.

Save Tree

Save the current tree as a .json file.


Preferences

Node Thickness

The thickness given to a node's outline displayed in the tree panel.

Node Radius

The radius (size) of nodes displayed in the tree panel.

Auto Update

Automatically save a node's content/description when edited.

Auto Update behaviour for internal versions

Determines how nodes with child versions are handled when auto update is enabled.

None - Do not update internal versions.
Save as New Version - Save any edits as a new child version of the current node.
Discard Child Versions - Remove all child versions of the current node and update the content and description.
Save Preferences

Saves the current preferences to be loaded upon reopening the program.
